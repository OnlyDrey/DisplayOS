# DisplayOS Branding Assets

This directory contains branding assets for DisplayOS.

## Files

- `logo-placeholder.png` - DisplayOS logo (replace with your own)
- Add additional branding files as needed

## Guidelines

### Logo
- Recommended size: 256x256 pixels
- Format: PNG with transparency
- Use for: Boot screen, about dialogs, documentation

### Wallpaper
- Location: `../wallpaper.png`
- Recommended size: 1920x1080 pixels
- Format: PNG or JPG
- Use for: Desktop background, login screen

### Favicon
- Recommended size: 32x32 pixels or ICO format
- Format: ICO or PNG
- Use for: Browser tabs (if hosting local content)

## Customization

Replace these placeholder files with your own branding before building the ISO.

See `docs/REBUILD_GUIDE.md` for detailed customization instructions.
